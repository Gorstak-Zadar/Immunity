# 🛡️ Immunity

> Registry tweak with **P3P (Platform for Privacy Preferences)** blocklist — blocks tracking/ad domains (180solutions, 2every, ad networks, etc.) in Internet Settings.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📋 **P3P History** | Populates `HKLM\...\Internet Settings\P3P\History` with block entries |
| 🚫 **Trackers** | Blocks 180solutions.com, yieldmanager, focalink, and many ad/tracker domains |
| 📝 **Registry Merge** | Double-click `.reg` to apply |

---

## 📋 Requirements

| Requirement | Details |
|-------------|---------|
| **OS** | Windows 10/11 |
| **Scope** | Current user (HKCU) |

---

## 🚀 Usage

1. **Create a restore point** (recommended)
2. Double-click `Immunity.reg` or right-click → **Merge**
